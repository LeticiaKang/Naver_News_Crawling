# Project_Naver-News-Crawling
네이버 뉴스 크롤링 응용프로그램 
  
## 프로젝트 목적
- 뉴스보는 습관을 기르기 위해 상위 10개의 뉴스를 크롤링하는 응용프로그램을 개발했습니다.
    
## 요구사항(주요 기능)
- 로그인 기능
- 뉴스 데이터 크롤링
- 사용자 친화적인 GUI 인터페이스로 구현

## 개발 환경 및 사용 기술
* Ptyhon : 프로젝트 주요 언어
* PyQt5 : GUI 애플리케이션 개발
* Selenium, BeautifulSoup : 크롤링, 데이터 추출
* ConfigParser : 설정 파일 및 로그인 정보 관리
