# Project_Naver-News-Crawling
네이버 뉴스 크롤링 응용프로그램 
  
## 프로젝트 목적
- 뉴스보는 습관을 기르기 위해 상위 10개의 뉴스를 크롤링하는 응용프로그램을 개발했습니다.  
    
## 요구사항
- 로그인 기능을 구현한다.(추후 카페에서 원하는 주제의 게시물을 크롤링하여 보여주는 프로그램을 만들기 위함)
- 뉴스 홈을 입력하면, 실시간 상위 10개의 뉴스를 보여준다.
